# Ruimte Spellen Pro 🚀

Adrian's ruimte-spellen, nu gebouwd met **Phaser 3** (game-engine) en
**Vite** (bouwtool) voor een veel professionelere uitstraling.

## Snel starten

```bash
npm install      # eenmalig: haalt de bouwstenen op
npm run dev      # start het spel (open de link die verschijnt)
```

Zie **INSTALLATIE.md** voor de volledige stap-voor-stap gids (Windows,
geen ervaring nodig).

## Wat zit erin?

- **Hoofdmenu** met geanimeerde sterrenhemel en regenboog-titel
- **Ballon Merge** — volledig uitgewerkt als voorbeeld, met 3D-ballonnen,
  vloeiende animaties, particles en camera-effecten

De andere spellen (rekenen, cijfers schrijven, piano, platformer, tikker)
kunnen op dezelfde manier worden overgezet — de structuur staat klaar.

## Mappen

- `src/main.js` — registreert alle spellen
- `src/scenes/` — elk spel is een aparte scene
- `public/` — voor toekomstige plaatjes en geluiden
